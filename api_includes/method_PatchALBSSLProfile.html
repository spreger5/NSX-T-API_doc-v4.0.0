Policy > Networking > Network Services > Advanced Load Balancing > Alb SSL Profiles

Create or update a ALBsslprofile (Deprecated)

If a ALBsslprofile with the alb-sslprofile-id is not already present,
create a new ALBsslprofile. If it already exists, update the
ALBsslprofile. This is a full replace.
This is a deprecated API. It is recommennded to use NSX Advanced Load Balancer (Avi) Controller UI or API directly instead of NSX-T ALB Policy UI and API.
This API is only available when using VMware NSX.
