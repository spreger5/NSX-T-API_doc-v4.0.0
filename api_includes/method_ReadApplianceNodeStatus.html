Read node status

Returns information about the node appliance's file system, CPU,
memory, disk usage, and uptime.

Request:

Method:
GET
URI Path(s):
/api/v1/transport-nodes/<transport-node-id>/node/status
/api/v1/node/status
/api/v1/cluster/<cluster-node-id>/node/status
Request Headers:
n/a
Query Parameters:
n/a
Request Body:
n/a

Example Request:

GET https://<nsx-mgr>/api/v1/node/status

Successful Response:

Response Code:
200 OK
Response Headers:
Content-type: application/json
Response Body:
