Cloud Service Manager > Azure > Security Groups

Returns information about a particular Azure security group

Returns information about a Azure security group like inbound rules,
outbound rules and priority.

Request:

Method:
GET
URI Path(s):

Request Headers:
n/a

n/a
Request Body:
n/a
