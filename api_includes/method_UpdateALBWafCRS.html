Policy > Networking > Network Services > Advanced Load Balancing > Alb Waf Crs

Create or update a ALBWafCRS (Deprecated)

If a ALBWafCRS with the alb-WafCRS-id is not
already present, create a new ALBWafCRS. If it already exists,
update the ALBWafCRS. This is a full replace.
This is a deprecated API. It is recommennded to use NSX Advanced Load Balancer (Avi) Controller UI or API directly instead of NSX-T ALB Policy UI and API.
This API is only available when using VMware NSX.
